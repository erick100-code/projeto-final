o que é um accordion.

na programação accordion é uma componente de interface usado frequentemente em landing pages para responder perguntas comuns, essa é sua função semantica

o que é semantica

semantica são elementos que tem significado com nav (area de navegação) e main (conteudo principal) do site

qual a diferença entre rem, em e px

px (pixels) são valores fixos, são unidades que compõem a largura de uma tela. em é o font size do elemento pai. rem é o font size do html que normalmente é 16 pixels ou seja, 2rem é igual a 16px + 16px, 2 rem = 32px

qual a diferença entre back e front end

o front end cuida da parte visual do site, da interface, ele é responsavel por tudo o que vemos. back end cuida da segurança e para onde os dados do site serão mandados obs:(banco de dados)

o que é um circulo na geometria

um círculo é uma figura plana formada por todos os pontos que estão a uma distância igual ou menor que um certo raio de um ponto central fixo

para que servem os menus em sites?

com minhas própias palavras eu digo, além de serem interativos e agradaveis visualmente, eles são uma parte importante de um site, onde ficarão conteudos para troca de tela em relação a um determinado tema ou assunto seja ele qual for. ele serve basicamente e intencionalmente propositalmente para a acessibilidade de um usuario permitindo que ele acesse diferentes conteudos sem ocupar sempre toda a estrutura.
